# Path to small image at top of user list

The login screen can optionally show a small image to provide site administrators and distributions a way to display branding.

- Type: dconf
- Key: /org/gnome/login-screen/logo
- Default for 22.04: '/usr/share/plymouth/ubuntu-logo.png'
- Default for 24.04: '/usr/share/plymouth/ubuntu-logo.png'
- Default for 25.10: '/usr/share/plymouth/ubuntu-logo.png'
- Default for 26.04: '/usr/share/pixmaps/ubuntu-logo-text-dark.svg'

Note: default system value is used for "Not Configured" and enforced if "Disabled".

Supported on Ubuntu 22.04, 24.04, 25.10, 26.04.

(ref::external)=
# External links and resources

ADSys depends on multiple tools and applications. It is not possible
for us to provide support and troubleshooting for each dependency.

The sections below include external links to relevant resources.

If you are having problems with one of these tools, and cannot find a solution
in the ADSys documentation, consult the external links below.
